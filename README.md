# Portfolio

Interactive portfolio site built with React, Vite, TypeScript, and Tailwind CSS.

This repo contains my personal portfolio site. I designed it as a single-screen experience, so instead of relying on long scrolling, the site uses buttons, panels, and transitions to move through content. The goal was to make the portfolio feel more like an interface than a document.

## Live Site

www.gatessubroto.com

## Overview

This site introduces who I am, the kind of work I do, and the projects I’ve built.

It highlights work across:
- interactive UI
- backend services
- data-heavy tools
- applied AI projects

The portfolio itself is also a project. I wanted it to reflect how I think: structured, interactive, and built around making information clear and usable.

## Features

- Single-screen portfolio layout
- Button-based navigation
- Project cards with expanded detail views
- Responsive design
- Animated transitions and hover states
- Built to showcase both engineering and interface work

## Tech Stack

- React
- Vite
- TypeScript
- Tailwind CSS

## Projects Featured

This portfolio currently includes projects such as:

- **Video Platform Demo**  
  A Node.js-based video platform with authentication, uploads, comments, and delete functionality.

- **Fair-Lending Disparity Analysis Tool**  
  An interactive analytics platform built around real HMDA and Census data, with filtering, drilldowns, mapping, ETL, and TensorFlow.js-based prediction.

- In Progress **AI Dental Insurance Coverage Assistant**  
  A proof-of-concept application using Java, Spring Boot, Angular, Docker, and the OpenAI API to answer insurance questions grounded in a plan document.

- In Progress **Claude-Powered Chatbot**  
  A lightweight chatbot built with React, Vite, Node.js, and Express.

- In Progress **Deep Learning Wellbeing Predictor**  
  A graduate deep learning project using TensorFlow, Keras, NumPy, and Pandas to predict self-reported health from lifestyle and eating data.

- **JavaScript Game / Portfolio Work**  
  Additional projects focused on UI, interaction, and frontend implementation.

## Why I Built It

I wanted a portfolio that showed more than a list of technologies or screenshots.

This site is meant to feel like a real interface: structured, intentional, and interactive. It reflects the kind of work I like doing — taking disconnected pieces, shaping them into something clear, and building tools people can actually use.

## Getting Started

Install dependencies:

```bash
npm install
```

Run:

```bash
npm run dev
```
